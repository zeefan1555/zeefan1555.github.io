const关键字 ::

const的作用： 被它修饰的值不能改变，是只读变量。必须在定义的时候就给它赋初值。

若要修改const修饰的变量的值，需要加上关键字[[volatile]]; 若想要修改const成员函数中某些与类状态无关的数据成员，可以使用mutable关键字来修饰这个数据成员；

1: 总结

[!tip]
左定向 (无法改变指向对象), 右定值 (无法改变指向对象的值)，
左右: 指的是 const 在 * 的左还是右边
顶层和底层也很好理解: 顶层说的是指针, 底层说的是数据

  • 常量指针是一个不能给改变指向的指针。顶层const

  • 指针是个常量，必须初始化.

    • 一旦初始化完成，它的值（也就是存放在指针中的地址）就不能在改变了，即不能中途改变指向，如int *const p。
      eg:
      const int* d=new int(2);, const 在 * 的左边,

  • 指针常量是一个指针，底层const

  • 读成常量的指针，指向一个只读变量，即底层数据无法通过指针改变

    • 也就是后面所指明的int const 和 const int，都是一个常量，可以写作int const *p或const int *p。

eg:
int *const e=new int(2);, const 在 * 的右边

1.1: 指针常量 (顶层const): 指针指向的对象无法改变

指针常量： 指针常量是指定义了一个指针，这个指针的值只能在定义时初始化，其他地方不能改变。
指针常量强调的是==指针==的不可改变性。💡不可以指向新的对象
特点： 靠近变量类型。

1.2: 常量指针（底层const) : 底层数据无法通过指针改变

常量指针： 是指定义了一个指针，这个指针指向一个只读的对象，不能通过常量指针来改变这个对象的值。
常量指针强调的是指针对其所指对象的不可改变性。💡不可以通过指针来改变对象
特点： 靠近变量名。
