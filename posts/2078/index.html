6.螺旋矩阵II

本文最后更新于：2022年9月13日 晚上

1 59.螺旋矩阵II

力扣题目链接

给定一个正整数 n，生成一个包含 1 到 n^2 所有元素，且元素按顺时针顺序螺旋排列的正方形矩阵。

示例:

输入: 3 输出: [ [ 1, 2, 3 ], [ 8, 9, 4 ], [ 7, 6, 5 ] ]

如果输入 4 , 则输出 4 * 4的矩阵, 输出的是正方形的螺旋矩阵.

2 思路::

大家还记得我们在这篇文章数组：每次遇到二分法，都是一看就会，一写就废
中讲解了二分法，提到如果要写出正确的二分法一定要坚持[[循环不变量原则]]。
循环不变量: 不变量是我们队每条边的一个处理规则

而求解本题依然是要坚持循环不变量原则。
不变量: 每次只处理第一个元素到最后元素的前一个

模拟顺时针画矩阵的过程:

  • 填充上行从左到右
  • 填充右列从上到下
  • 填充下行从右到左
  • 填充左列从下到上

由外向内一圈一圈这么画下去。

3 代码

v1

class Solution {
public:
    vector<vector<int>> generateMatrix(int n) {
        vector<vector<int>> res(n, vector<int>(n, 0));
        int startx = 0, starty = 0;
        int mid  = n / 2;
        int count = 1;
        int offset = 1; // 用来控制长度
        int loop  = n / 2; 
        while(loop--){
            int i; // 设置成全局变量, 因为下面也要用
            int j; 
            for(j = startx; j < n - offset; j++){
                res[startx][j] = count;// 可以成一行 count++
                count++; 
            }
            for(i = starty; i< n - offset; i++){// 💡上面 j已经+1 跳出了循环. 已经在边界了
                res[i][j] = count++; 
            }
            for(; j > startx; j--){// 要大于 startx 而不是 0, 因为下一层循环还要用
                res[i][j] = count++; 
            } 
            for(; i > starty; i--){
                res[i][j] = count++; 
            }
            startx++; 
            starty++; 
            offset += 1; 
        }
         // 如果n为奇数的话，需要单独给矩阵最中间的位置赋值
        if (n % 2 == 1) {
            res[mid][mid] = count;
        }
        return res;
    }```

```c++

class Solution {
public:
    vector<vector<int>> generateMatrix(int n) {
        vector<vector<int>> res(n, vector(n, 0));
        int startx = 0, starty = 0;
        int mid = n/2;
        int count = 1;
        int offset = 1;
        int loop = n/2;
        // eg 3 * 3 矩阵
        while(loop--){
            int i , j ;
            for(j = starty; j < n - offset; j++){
                res[startx][j] = count++; // i: 第一行, j: 0 → 1 列 
            }
            for(i = startx; i < n - offset; i++){
                res[i][j] = count++; // i: 0 → 1 行, j: 第2 列
            }
            for(; j > starty; j--){
                res[i][j] = count++; // i:第 2 行, j: 2 → 1 列
            }
            for(; i  > startx; i--){
                res[i][j] = count++;// i: 2 → 1 行, j: 0 列

            }
            startx++;
            starty++;
            offset++;
        }
        if(n % 2 == 1){
            res[mid][mid] = count;
        }
        return res;

    }
};```
