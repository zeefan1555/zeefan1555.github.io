2 数组求和

for (循环变量类型 循环变量名称 : 要被遍历的对象) 循环体
应用：对数组求和
从第一个元素开始操作

//A是数组，for循环对数组求和
int result = 0;
for (int a : A)  result += a; // a没有特殊的意义，只是指明循环变量的类型，“:”后只需数组名即可
